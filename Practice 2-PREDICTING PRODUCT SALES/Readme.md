# 0. Problem Definition

## Project Objective

The objective of this project is to predict the total sales amount of customer orders using customer behavior, product information, and transaction data collected from an e-commerce platform. Accurate sales prediction helps businesses optimize inventory management, improve demand forecasting, reduce overstock and stockout situations, and develop more effective marketing strategies.

---

## Problem Type

This project belongs to **Supervised Learning – Regression**.

The target variable is **Total_Amount**, which represents the total sales amount of each customer order. Since the target is a continuous numerical value, the task is formulated as a regression problem.

The model learns to predict **Total_Amount** based on features such as:

- Customer demographics (Age, Gender, City)
- Product information (Product Category, Unit Price)
- Purchase behavior (Quantity, Discount Amount)
- Transaction information (Payment Method, Device Type)
- Customer engagement (Session Duration, Pages Viewed)
- Customer experience (Delivery Time, Customer Rating)
